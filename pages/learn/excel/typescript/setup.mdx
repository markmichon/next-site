### Download Starter Code (Optional)

If you’re NOT continuing from the basics lessons, you can download, install, and run the starter code for this lesson below. This will set up a `nextjs-blog` directory such that it’s identical to the result of the basics lessons, and then starts the development server.

Again, this is NOT necessary if you’ve just finished the basics lessons.

```bash
# If you prefer Yarn, you can use `yarn create` instead of `npm init`
npm init next-app nextjs-blog --example "https://github.com/zeit/next-learn-starter/tree/master/basics-final"
cd nextjs-blog
# If you prefer Yarn, you can use `yarn dev` instead
npm run dev
```

Once you download your starter code, you should update:

- `public/images/profile.jpg` with your photo (Recommended: 400px width/height).
- `const name = '[Your Name]'` in `components/layout.js` with your name.
- `<p>[Your Self Introduction]</p>` in `pages/index.js` with your self introduction.
